## ⭐️ Projeto Principal / Destaque Técnico

### 📑 Sistema de Análise SPED Contábil — ECD
* **Descrição:** Aplicação web de alta complexidade técnica voltada para a análise, leitura de registos, interpretação de dados estruturados e validação fiscal de arquivos do SPED Contábil (ECD). O sistema permite consultar todas as tabelas oficiais de acordo com o Manual da ECD.
* **Tecnologias:** PHP, MySQL, HTML, CSS, JavaScript, Bootstrap.
* **🔗 [Ver Repositório do Projeto](https://github.com/matheusaraujo019/sistema-sped-contabil)**
